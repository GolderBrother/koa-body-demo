## 二、运行说明：

### 1、克隆文件

```bash
git clone https://github.com/GolderBrother/koa-body-demo.git
```

### 2、安装依赖

```bash
yarn install
# npm install 
```

### 3、开启 `app.js`

> 支持 `nodemon`，`nodemon app.js`

```bash
node app.js
```

### 4、访问页面：

`http://127.0.0.1:3001`
